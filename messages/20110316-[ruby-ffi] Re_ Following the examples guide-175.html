Just a guess but to me it looks like you aren't specifying that the
output of the compilation step is a shared object.  Normally what gcc
produces is an object file which needs to be linked together into an
executable, static library or dynamic library.  The gcc man page says
it can generate a shared object, but I have never used it to do that
with "-shared" or "-shared-libgcc".  Otherwise you will need to use
"ld" and tell it to generate a shared library.
