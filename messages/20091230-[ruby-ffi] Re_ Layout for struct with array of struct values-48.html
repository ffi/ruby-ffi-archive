As an FYI, this raises a TypeError ('wrong argument type Class
(expected Data)'):

      class Field < FFI::Struct
        layout :type, :ushort, :value, :pointer
      end
      class Multifield < FFI::Struct
        layout :theFields, [Field, 1]
      end
